# agents

This repository contains a small Codex configuration bundle under `codex/`.
It defines local agent guidance, runtime settings, role-specific configs, and a bundled theme.

## Structure

- `codex/AGENTS.md`: local orchestration and write-safety rules
- `codex/config.toml`: main Codex runtime configuration
- `codex/agents/`: role-specific agent configs such as `worker`, `explorer`, and `reviewer`
- `codex/themes/`: bundled editor/theme assets

## Usage

Use this repo as a local Codex setup reference or starting point, and update the files under `codex/` to match your workflow.
